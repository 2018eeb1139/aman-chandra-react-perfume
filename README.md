# Aman's Perfume

A responsive web application for Aman's Perfumes built with React, Vite, Tailwind CSS, and shadcn/ui components.

## 📱 Features

- **Hero Section**: Eye-catching hero section with call-to-action
- **Product Showcase**: Display perfumes with images, descriptions, and pricing
- **Product Details**: Detailed view of individual perfumes with specifications
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Built with Tailwind CSS and shadcn/ui components
- **Error Handling**: Comprehensive error handling when perfume not found

## 🛠 Tech Stack

- **Frontend**: React 18 with JavaScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Routing**: React Router DOM
- **Icons**: Lucide React

## Installation & Setup

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Steps

1. **Clone the repository**

   ```bash
   git clone https://github.com/2018eeb1139/aman-chandra-react-perfume.git
   cd aman-chandra-react-perfume
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the URL shown in your terminal)

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
